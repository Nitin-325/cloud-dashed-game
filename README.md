# cloud-dashed-game
An interactive browser game where players click falling clouds to gain points while avoiding negative clouds. Built using HTML, CSS, and JavaScript.


☁️ Cloud Dashed Game :-
Cloud Dashed Game is a simple and interactive browser game where players click on falling clouds to gain points while avoiding negative clouds. The game includes a scoring system, lives tracking, and increasing difficulty to make gameplay more challenging and fun.


🎮 Features Of Game :-
🎮 Interactive cloud clicking gameplay
⭐ Different colored clouds with different points
❤️ Lives system
📊 Score tracking
⚡ Increasing game difficulty over time
🔄 Play again, resume, and exit options


🕹️ How to Play :-
Click the Play Game button to start.
Clouds will start falling from the sky.
Click on positive clouds to gain points.
Avoid negative clouds, as they reduce your score.
If you miss positive clouds or click the wrong ones, you may lose lives.
The game ends when all lives are lost.


🛠️ Technologies Used :-
HTML5
CSS3
JavaScript (DOM Manipulation & Game Logic)
